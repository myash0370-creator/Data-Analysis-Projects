# Page Recommendation System

A Python-based project that cleans and processes user data to generate basic page recommendations.

---

## 📌 Features
- Cleans messy/raw JSON data
- Removes invalid or missing values
- Converts unstructured data into readable format
- Prepares data for recommendation logic

---

## 🛠️ Technologies Used
- Python
- JSON

---

## 📂 Project Files
- `data.json` → Raw data  
- `data_2.json` → Processed data  
- `cleaned_data_2.json` → Final cleaned dataset  
- `dump_data.ipynb` → Raw data processing  
- `cleaning_data.ipynb` → Data cleaning logic  

---

## ▶️ How to Run
1. Open notebooks in Jupyter (`.ipynb` files)  
2. Run all cells  
3. View cleaned data output  

---

## 📊 What It Does
- Converts messy dump data into structured format  
- Cleans user-related information  
- Prepares dataset for recommendation systems  

---

## 🎯 Purpose
This project demonstrates:
- Data cleaning and preprocessing  
- Handling messy datasets  
- Preparing data for recommendation systems
